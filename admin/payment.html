Payment Setting

Mode Pembayaran Aktif

Pilih metode yang akan otomatis aktif di layar pembayaran:

QRIS String Statis

Paste QRIS string dari merchant. Sistem generate QR dinamis sesuai nominal. Konfirmasi manual oleh admin.

Buka aplikasi bank → menu QRIS → Salin string → paste di sini.

Duitku Payment Gateway

Pembayaran otomatis via Duitku (QRIS, VA, e-wallet, dll). Terdeteksi tanpa konfirmasi admin.

Daftar di sandbox.duitku.com → Project → salin Merchant Code & API Key.
Pastikan Callback URL di dashboard Duitku sudah diisi dengan endpoint Edge Function kamu.
