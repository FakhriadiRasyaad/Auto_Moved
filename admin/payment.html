Mode QRIS

Pilih cara QRIS bekerja:

QRIS String Statis

Paste QRIS string dari merchant. Sistem generate QR dinamis sesuai nominal. Konfirmasi manual oleh admin.

Buka aplikasi bank → menu QRIS → Salin string → paste di sini.

Midtrans Payment Gateway

Pembayaran otomatis terdeteksi tanpa konfirmasi admin.

Daftar di midtrans.com → Settings → Access Keys

Xendit Payment Gateway

Pembayaran otomatis terdeteksi tanpa konfirmasi admin.

Daftar di xendit.co → Settings → API Keys
